# 开源发布推进记录

## 当前定位

当前版本定位为：

- 可公开展示的原型版本
- 可供 Linux.do 用户手动安装和使用
- 重点强调“稳定、可解释、低风险”

## 当前使用闭环

1. 启动本地桥
2. 浏览器加载 `browser-extension`
3. 打开 Linux.do 帖子页
4. 点击“导出当前帖子”
5. 在扩展里查看阶段进度
6. 导出完成后打开结果目录
